# 手机维修订单管理系统建设计划

## 项目概述
根据提供的系统说明文档，构建一个包含客户端、技术员端和管理员端的全功能手机维修订单管理系统。

## 主要功能模块
- **客户端模块**：首页展示、订单查询、在线报价、客户服务
- **技术员端模块**：工作台、订单管理、客户沟通、维修记录、配件管理  
- **管理员端模块**：数据仪表板、订单管理中心、客户关系管理、库存管理、供应商管理、人员管理

## 技术栈
- 前端：Next.js 14 + TypeScript + Tailwind CSS
- 数据库：Prisma + PostgreSQL  
- 认证：NextAuth.js
- UI组件：Radix UI + Lucide React
- 状态管理：Zustand
- 表单：React Hook Form + Zod
